C* 控制律 — 567三回路改造方案（已确认接口）

基于 PitchNormalLaw.cpp L1711-1718 和 A320姿态控制器说明 §3.1

当前 vs 改造后

旧 (三回路级联):  nz_err → ∫ → α_err × Kα → q_err × Kw → de
新 (C* 单回路):  C*_err → [PID] → de   q 作为阻尼前馈并联注入，不串级

1. C* 被控量计算 (§3.1 + PitchNormalLaw L1503-1507)

Nz_comp = Nz / cos(θ) / cos(φ)   补偿俯仰和滚转角
Vm = lookup(Vc_kn, Vm_table)   1-D 查表，空速→交叉速度
C*_meas = Nz_comp + (Vm / g) × q
C*_dem = nz_cmd   从制导来的法向过载指令
项含义来源取值
Nz法向过载 (g)已有输入 nz端口 u2
θ, φ俯仰/滚转角From 块读取需新增 2 个 From
VmC* 交叉速度PitchNormalLaw L15061-D Lookup f(Vc), 见下表
g重力常数Constants9.80665
q俯仰角速率已有输入 q端口 u3
nz_cmd法向过载指令已有输入 nz_cmd端口 u1

2. Vm 查表数据（PitchNormalLaw.cpp L1506-1507）

Vc (kn)050100150200250300400
Vm (kn)050100150200250300400

简化为 Vm_kn = Vc_kn（1:1映射）。需转换成 m/s: ×0.5144。或用你飞机的速度量级自定义断点。

3. PID 控制律（PitchNormalLaw L1711-1718）

e_C* = C*_dem − C*_meas
de_raw = Kp(Vc) × e_C* + Kq(Vc) × q + Ki × ∫e_C*·dt
项作用取值方式初值建议
Kp(Vc)C*误差→舵面比例1-D 查表 f(Vc)，Vc↑→Kp↓
因动压增大舵效增强		Vc=200m/s: Kp≈1.5 deg/e_C*
Vc=100m/s: Kp≈4.0 deg/e_C*
Kq(Vc)俯仰率阻尼1-D 查表 f(Vc)
高速高阻尼防振荡		Vc=200m/s: Kq≈0.8 deg/(deg/s)
Vc=100m/s: Kq≈0.5 deg/(deg/s)
Ki自动配平按构型切换常数
光洁0.3°/s 襟翼0.7°/s 地面0.7°/s		0.3 deg/s（光洁巡航）

注：A320 的 D 项不走 e_C* 的导数，直接用 q 反馈替代（Kq(Vc)·q）。这避免了微分噪声放大。
