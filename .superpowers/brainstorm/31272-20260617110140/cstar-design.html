A320 C* 控制律 — 567三回路子系统改造方案

基于 PitchNormalLaw.cpp 和 A320_姿态控制器阶段与结构说明.md

1. 核心公式：C* 被控量

C* = Nz + (Vm / g) * q
法向过载 + 交叉速度/重力 × 俯仰率 → 混合被控量
项含义来源取值方式
Nz法向过载 (g)IMU/加速度计模型已有，来自 nz 信号
VmC* 交叉速度 (m/s)PitchNormalLaw.cpp §Vm_currentms查表 f(Vc)，典型值 150~200 m/s
g重力加速度常量9.80665 m/s²
q俯仰角速率 (rad/s)IMU模型已有

物理意义：低速时 q 主导（类似俯仰率指令），高速时 Nz 主导（类似过载指令）。Vm 控制两者的"切换速度"。

2. 顶层信号流

输入: nz_cmd(制导), nz(传感器), q(传感器), Vc(空速), Flight_Phase(状态机), θ, φ
输出: de_cmd_deg (升降舵指令 deg)

nz_cmd ──→ [C*_dem] ──→ ⓔ ──→ [Kp(Vc)] ─┐
                                  │
Nz ──→ ╮                           ├─Ⓢ────→ [RateLimiter] → de_cmd
q ───→ [Vm/g] ──→ Ⓢ → C*_meas ──┘        │              ↑
                        q ──→ [Kq(Vc)] ──┤              ↑
                                      │              ↑
                       ∫e·dt ─→ [Ki(Vc)] ──┤  (←自动配平通道)

3. 逐项详解

3.1 C*_dem — 需求值生成

C*_dem = nz_cmd   (制导给出的法向过载指令)
注：A320 有人驾驶时侧杆位置 → C*_dem 查表。无人机可从制导直接取 nz_cmd

3.2 C*_meas — 测量值计算

C*_meas = Nz + (Vm/g) × q
其中 Nz 需补偿俯仰角影响：Nz_comp = Nz / cos(θ) / cos(φ)
参数来源取值
Nz_compPitchNormalLaw L1503Nz_g - 1/cos(θ)·cos(φ) → 补偿重力分量
VmPitchNormalLaw L1506-1507查表 look1_binlxpw(V_tas_kn, bp, table)
断点: [0, 50, 100, 150, 200, 250, 300, 400] ks
值: [0, 50, 100, 150, 200, 250, 300, 400] → 1-D插值
g常量9.80665

3.3 PID 结构

de_raw = Kp(Vc) × e_C* + Kd(Vc) × q + Ki(Vc) × ∫e_C*·dt
项物理意义来源取值方式
Kp(Vc) 比例增益（C*误差→舵面） PitchNormalLaw §PLUT 1-D 查表 f(Vc_kn)
Vc↑ → Kp↓（动压增大，舵效增强）
典型范围：0.5~2.0 deg/e_C*
Kd(Vc) 俯仰率阻尼增益 PitchNormalLaw §DLUT 1-D 查表 f(Vc_kn)
直接用 q 反馈而非 e_C* 的导数
A320 把 D 项放在 q 通道，不放在 C* 误差通道
Ki(Vc) 积分增益（自动配平） PitchNormalLaw §自动配平速率 按飞行构型切换
光洁: 0.3 deg/s   襟翼: 0.7 deg/s   地面: 0.7 deg/s

3.4 增益表数据（来自 PitchNormalLaw.cpp）

表名断点 Vc_kn表值用途

Loaddemandθ_err: [-1, 0, 6, 11, 15] deg[1, 1, 0.5, 0.3, 0.3]大俯仰误差时限幅
