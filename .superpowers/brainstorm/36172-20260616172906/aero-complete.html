A320 完整气动模型

所有插值表 + 单点常数 + 构型增量 + 计算公式

0. 无量纲角速率定义

p̂ = p · b_ref / (2·Va)    无量纲滚转率
q̂ = q · c_ref / (2·Va)    无量纲俯仰率
r̂ = r · b_ref / (2·Va)    无量纲偏航率
Va = 真空速, 防止 Va→0 时分母取 V_min=3 m/s

1. 升力系数 CL

CL_base = interp1(α_CL_table, CL_table, α, 'pchip')
→ §7, 9节点, α ∈ [-3.15, 3.15] rad

K_de = interp1(α_Kde_table, Kde_table, α, 'pchip')
→ §9, 11节点, 升降舵效能倍率, 大迎角衰减→反向

g_L_TE = interp1(δf_TE_table, gL_TE_table, δf_TE, 'pchip')
→ §13.1, 6节点, 后缘襟翼偏角→升力比例, δf_TE ∈ [0, 0.698] rad

g_L_LE = interp1(δs_LE_table, gL_LE_table, δs_LE, 'pchip')
→ §13.2, 6节点, 前缘缝翼偏角→升力比例, δs_LE ∈ [0, 0.471] rad

η_sym = (δ_spL + δ_spR) / (2·δ_sp_max)  对称扰流板比例, §17

CL = CL_base
  + CLq·q̂  俯仰率→升力 §10
  + CLde·K_de·δe  升降舵→升力 §9+§10
  + ΔCL_flap_tot·[g_L_TE + 0.01·g_L_LE]  襟翼增升 §6+§13
  + ΔCL_sp·η_sym  扰流板卸升 §17

2. 阻力系数 CD

注意: CD 依赖 CL (诱导阻力项), 必须先算 CL 再算 CD

g_D_TE = interp1(δf_TE_table, gD_TE_table, δf_TE, 'pchip')
→ §13.1, 6节点

g_D_LE = interp1(δs_LE_table, gD_LE_table, δs_LE, 'pchip')
→ §13.2, 6节点

CD = CD0  零升阻力 §6
  + k_i_eff·(CL − CL_D0)²  诱导阻力 §4.1
  + ΔCD_flap_tot·[g_D_TE + 0.5·g_D_LE]  襟翼增阻 §6+§13
  + ΔCD_sp·η_sym  扰流板增阻 §17
  + ΔCD_gear·η_gear  起落架增阻 §6+§18

3. 俯仰力矩系数 Cm

Cm_base = interp1(α_Cm_table, Cm_table, α, 'pchip')
→ §8, 14节点, ⚠️ 符号依赖MSFS坐标系

Cm = Cm_base
  + Cmq·q̂  俯仰阻尼 §10
  + Cmde·K_de·δe  升降舵→俯仰力矩 §9
  + ΔCm_flap_tot·[δf_TE/δf_TE_max + δs_LE/δs_LE_max]  襟翼力矩 §6+§13
  + ΔCm_sp·η_sym  扰流板力矩 §17
  + ΔCm_gear·η_gear  起落架力矩 §6+§18

4. 侧力系数 CY

CY = CSb·β  侧滑侧力 §10
  + CSr·r̂  偏航率侧力 §10
  + CSdr·δr  方向舵侧力 §10

5. 滚转力矩系数 Cl

Cl = Clb·β  上反效应 §10
  + Clp·p̂  滚转阻尼 §10
  + Clr·r̂  偏航率→滚转耦合 §10
  + Clda·δa  副翼滚转 §10
  + Cldr·δr  方向舵→滚转耦合 §10

6. 偏航力矩系数 Cn

Cn = Cnb·β  风标稳定 §10
  + Cnp·p̂  滚转率→偏航耦合 §10
  + Cnr·r̂  偏航阻尼 §10
  + Cnda·δa  不利偏航 §10
  + Cndr·δr  方向舵偏航 §10
