当前 basedata.m 中所有气动量：定义方式、来源、用法

总气动模型

CL = CL_table(α) + CLde · K_δe(α) · δe + ΔCL_flap · flap_ratio
CD = CD0 + k_i_eff · (CL − CL_D0)² + ΔCD_flap · flap_ratio
Cm = Cm_table(α) + Cmde · K_δe(α) · δe + ΔCm_flap · flap_ratio
CY = CSb·β + CSr·r̂ + CSdr·δr
Cl = Clb·β + Clp·p̂ + Clr·r̂ + Clda·δa + Cldr·δr
Cn = Cnb·β + Cnp·p̂ + Cnr·r̂ + Cnda·δa + Cndr·δr

CL 完整计算（Simulink 中应实现为）

CL_base = interp1(alpha_CL_table, CL_table, α, 'pchip')
K_de = interp1(alpha_Kde_table, Kde_table, α, 'pchip')
ΔCL_flap_eff = ΔCL_flap * flap_ratio % flap_ratio ∈ [0,1]
CL = CL_base + CLde * K_de * δe + ΔCL_flap_eff

Cm 完整计算

Cm_base = interp1(alpha_Cm_table, Cm_table, α, 'pchip')
K_de = interp1(alpha_Kde_table, Kde_table, α, 'pchip')
ΔCm_flap_eff = ΔCm_flap * flap_ratio
Cm = Cm_base + Cmde * K_de * δe + ΔCm_flap_eff

CD 完整计算

CD = CD0 + K*(CL − CL_D0)² + ΔCD_flap*flap_ratio
% 注意：CD 依赖 CL（因为诱导阻力），必须先算 CL 再算 CD
