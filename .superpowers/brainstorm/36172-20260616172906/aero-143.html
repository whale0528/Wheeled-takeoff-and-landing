§14.3 总气动系数组装（修正版）

严格按文档公式。K_δe(α) 仅在俯仰力矩中使用，升力不用。

1. 升力系数 CL

CL = K_GE·[
  K_CL,cruise·CL_table(α)
  + s_Lq·CLq·q̂  俯仰率升力 §10
  + s_Lδe·CLde·δe  升降舵升力 §10, 不乘 K_δe
]
+ ΔCL_flap + ΔCL_sp

2. 阻力系数 CD

CD = CD0
+ ΔCD0(M)
+ k_i_eff·(CL − CL_D0_config)²
+ ΔCD_flap + ΔCD_gear + ΔCD_sp + ΔCD_control

3. 俯仰力矩系数 Cm   ← K_δe(α) 只在这里

Cm =
s_mα·Cm_table(α)
+ s_mq·Cmq·q̂  俯仰阻尼 §10
+ s_mδe·Cmde·K_δe(α)·δe  升降舵力矩 §9, 只有Cm乘了K_δe
+ ΔCm_flap + ΔCm_gear + ΔCm_sp
+ Cm0,trim

4. 侧力系数 CY

CY =
s_Yβ·CYβ·β
+ s_Yp·CYp·p̂  滚转率侧力 §10, 值=1.833
+ s_Yr·CYr·r̂
+ s_Yδr·CYδr·δr

5. 滚转力矩系数 Cl

Cl =
s_lβ·Clβ·β
+ s_lp·Clp·p̂
+ s_lr·Clr·r̂
+ s_lδa·Clδa·δa
+ s_lδr·Clδr·δr
+ ΔCl_asym

6. 偏航力矩系数 Cn

Cn =
s_nβ·Cnβ·β
+ s_np·Cnp·p̂
+ s_nr·Cnr·r̂
+ s_nδa·Cnδa·δa
+ s_nδr·Cnδr·δr
+ ΔCn_asym

7. 构型增量

襟翼 (§6+§13) — 插值表查 g_L, g_D

g_L_TE = interp1(δf_TE_tab, gL_TE_tab, δf_TE, 'pchip')  §13.1
g_D_TE = interp1(δf_TE_tab, gD_TE_tab, δf_TE, 'pchip')
g_L_LE = interp1(δs_LE_tab, gL_LE_tab, δs_LE, 'pchip')  §13.2
g_D_LE = interp1(δs_LE_tab, gD_LE_tab, δs_LE, 'pchip')

ΔCL_flap = 1.867·[g_L_TE + 0.01·g_L_LE]
ΔCD_flap = 0.1316·[g_D_TE + 0.5·g_D_LE]
ΔCm_flap = −0.084·[δf_TE/δf_TE_max + δs_LE/δs_LE_max]

扰流板 (§17)

η_sym = (δ_spL + δ_spR) / (2·δ_sp_max)
ΔCL_sp = −0.466875·η_sym
ΔCD_sp = 0.05775·η_sym
ΔCm_sp = 0.023·η_sym

起落架 (§17+§18)

η_gear ∈ [0,1]
ΔCD_gear = 0.0372·η_gear
ΔCm_gear = 0.0022·η_gear
