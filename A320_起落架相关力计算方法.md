# A320 起落架相关力计算方法

本文说明如何使用已提取的 A320 起落架参数计算起飞、着陆、滑跑阶段的起落架力。坐标系按你的要求定义为：

```text
机体系 b:
x 轴：机头向前
y 轴：机体右侧
z 轴：向下
```

这是右手坐标系，满足 `x × y = z`。因此地面对飞机的支反力方向通常是 `-z` 方向，但严格计算时应沿地面上法线，而不是简单固定为机体系 `-z`。

这些刚度和阻尼是按最大起飞重量静载荷分配推导的等效线性参数。它们适合起飞滑跑、接地、着陆滑跑和正常刹车减速仿真。

## 2. 每帧计算总流程

每个仿真步长内按以下顺序计算：

1. 将每个起落架接地点从机体系转换到地面/惯性系。
2. 判断接地点是否穿入地面，得到压缩量 `delta_i`。
3. 计算接地点速度，得到压缩速度 `delta_dot_i`。
4. 用弹簧阻尼计算支反力 `N_i`。
5. 计算接地点相对地面的切向速度。
6. 在轮胎纵向/侧向坐标中计算滚动阻力、刹车力、侧向力。
7. 用摩擦圆或摩擦椭圆限制纵向和侧向合力。
8. 把每个接地点的力转换到机体系，求和得到全机起落架力和力矩。

## 3. 接地点位置计算

设第 `i` 个起落架接地点在机体系中的固定位置为：

```text
r_i^b = [x_i, y_i, z_i]^T
```

飞机质心在地面系中的位置为：

```text
p_CG^g
```

机体系到地面系的方向余弦矩阵为：

```text
R_b^g
```

则接地点在地面系中的位置：

```text
p_i^g = p_CG^g + R_b^g * r_i^b
```

若你的地面系也采用 `z` 向下，并令平直跑道地面为：

```text
z_ground = 0
```

则飞机在地面上方时 `z_CG^g < 0`，接地点穿入地面时 `p_i,z^g > 0`。

## 4. 接触判断与轮胎压缩量

### 4.1 平直跑道

平直跑道、地面系 `z` 向下时：

```text
h_i = p_i,z^g - z_ground
```

其中：

| 条件 | 含义 |
|---|---|
| `h_i <= 0` | 接地点在地面上方或刚好接触，起落架不受压 |
| `h_i > 0` | 接地点穿入地面，起落架被压缩 |

压缩量：

```text
delta_i = clamp(h_i, 0, delta_max_i)
```

### 4.2 任意地面法线

若跑道有坡度，设地面上法线为 `n_up^g`，且 `n_up^g` 指向天空。接地点到地面的有符号高度：

```text
s_i = n_up^g · (p_i^g - p_ground^g)
```

其中 `s_i > 0` 表示点在地面上方，`s_i < 0` 表示点穿入地面。压缩量：

```text
delta_i = clamp(-s_i, 0, delta_max_i)
```

这个形式更通用，推荐工程实现时使用。

## 5. 接地点速度与压缩速度

飞机质心速度为 `v_CG^g`，角速度为 `omega^g`，则接地点速度：

```text
v_i^g = v_CG^g + omega^g × (R_b^g * r_i^b)
```

平直跑道、地面系 `z` 向下时，压缩速度：

```text
delta_dot_i = v_i,z^g
```

含义：

| `delta_dot_i` | 含义 |
|---:|---|
| `> 0` | 接地点向下运动，起落架正在压缩 |
| `< 0` | 接地点向上运动，起落架正在伸长 |

任意地面法线形式：

```text
delta_dot_i = - n_up^g · v_i^g
```

如果没有接触，即 `delta_i = 0` 且 `delta_dot_i <= 0`，则该起落架不产生地面力。

## 6. 支反力计算

### 6.1 线性弹簧阻尼

推荐先用线性弹簧阻尼模型：

```text
N_raw_i = k_i * delta_i + c_i * delta_dot_i
N_i = max(0, N_raw_i)
```

其中：

| 符号 | 含义 |
|---|---|
| `N_i` | 第 i 个起落架的法向支反力，单位 N |
| `k_i` | 起落架等效垂向刚度 |
| `c_i` | 起落架等效垂向阻尼 |
| `delta_i` | 压缩量 |
| `delta_dot_i` | 压缩速度 |

必须使用 `max(0, ...)`，因为地面只能推飞机，不能把飞机向下拉住。

### 6.2 法向力方向

任意地面法线下，地面对飞机的法向力：

```text
F_N_i^g = N_i * n_up^g
```

若采用平直跑道且地面系 `z` 向下：

```text
n_up^g = [0, 0, -1]^T
F_N_i^g = [0, 0, -N_i]^T
```

转到机体系：

```text
F_N_i^b = R_g^b * F_N_i^g
```

只有在平直跑道、小滚转/小俯仰角近似下，才可以直接写：

```text
F_N_i^b ≈ [0, 0, -N_i]^T
```

## 7. 轮胎纵向和侧向坐标

接地点处的摩擦力必须在地面切平面内计算。

地面上法线为 `n_up^g`，则切向速度：

```text
v_t_i^g = v_i^g - (v_i^g · n_up^g) * n_up^g
```

为每个轮子定义轮胎纵向方向 `e_xw_i` 和侧向方向 `e_yw_i`。

主起落架不转向：

```text
e_xw_main^b = [1, 0, 0]^T
e_yw_main^b = [0, 1, 0]^T
```

前起落架有转向角 `delta_s`，正方向取向右转：

```text
e_xw_nose^b = [cos(delta_s), sin(delta_s), 0]^T
e_yw_nose^b = [-sin(delta_s), cos(delta_s), 0]^T
```

把轮胎方向转到地面系并投影到地面切平面：

```text
e_xw_i^g = R_b^g * e_xw_i^b
e_xw_i^g = normalize(e_xw_i^g - (e_xw_i^g · n_up^g) * n_up^g)

e_yw_i^g = n_up^g × e_xw_i^g
```

轮胎纵向/侧向速度：

```text
u_i = v_t_i^g · e_xw_i^g
v_i = v_t_i^g · e_yw_i^g
```

其中 `u_i` 用于纵向滚动阻力和刹车力，`v_i` 用于侧向力。

## 8. 前轮转向限制

源文件前轮最大转向角为 `1.658063 rad`，约 95 deg。这个值适合低速滑行和 MSFS 配置，不建议高速起飞/着陆直接使用。

建议按速度限制前轮转向：

```text
delta_s = clamp(delta_cmd, -delta_lim(V), delta_lim(V))
```

工程建议：

| 地速 | 前轮最大转向角 |
|---|---:|
| 0 到 20 m/s | 1.309 rad，约 75 deg |
| 20 到 70 m/s | 从 1.309 rad 线性衰减到 0.105 rad |
| 大于 70 m/s | 0.105 rad，约 6 deg |

这样更符合真实飞机起飞和着陆高速滑跑要求，避免高速大转角导致不现实的侧向力。

## 9. 纵向力计算

纵向力包括滚动阻力和刹车力。

### 9.1 滚动阻力

源码没有给滚动阻力系数，独立仿真需要自定。干燥硬质跑道可先取：

```text
mu_roll = 0.015
```

滚动阻力：

```text
F_roll_i = - mu_roll * N_i * smooth_sign(u_i)
```

`smooth_sign` 建议用连续函数，避免低速振荡：

```text
smooth_sign(u) = u / sqrt(u^2 + eps_v^2)
```

推荐：

```text
eps_v = 0.2 m/s
```

### 9.2 刹车力

前起落架不刹车：

```text
B_nose = 0
```

左主起落架和右主起落架：

```text
B_left  = brake_left
B_right = brake_right
```

其中 `B` 为刹车力因子，范围 0 到 1。

若你不模拟液压压力，直接使用刹车指令：

```text
B_left  = clamp(0.518 * brake_left_cmd,  0, 1)
B_right = clamp(0.518 * brake_right_cmd, 0, 1)
```

这里 `0.518` 来自 `toe_brakes_scale`。

若你模拟 A32NX 刹车压力，可用源码中的刹车力因子：

```text
B = clamp(50 * (sqrt(P_norm_psi) + sqrt(P_altn_psi)) / 2538, 0, 1)
```

压力限制：

| 工况 | 压力限制 |
|---|---:|
| 正常刹车 | 2538 psi |
| 停留刹车，无人工输入 | 2103 psi |
| 备用刹车，防滞关闭 | 1160 psi |
| 备用刹车，防滞可用或人工输入 | 2538 psi |

刹车纵向力：

```text
F_brake_i = - mu_brake * B_i * N_i * smooth_sign(u_i)
```

干燥跑道可先取：

```text
mu_brake = 0.65
```

## 10. 侧向力计算

源码没有给轮胎侧偏刚度和 Pacejka 参数，因此不建议构造复杂轮胎模型。简单且稳定的侧向力模型：

```text
F_lat_raw_i = - mu_lat * N_i * sat(v_i / V_lat_ref, -1, 1)
```

推荐初值：

```text
mu_lat = 0.75
V_lat_ref = 2.0 m/s
```

含义：

- 小侧向速度时，侧向力近似线性。
- 侧向速度较大时，侧向力不会超过 `mu_lat * N_i`。
- 这满足起飞滑跑方向保持、接地侧偏修正和着陆滑跑侧向稳定的基本要求。

如果要更接近轮胎侧偏角模型，可改成：

```text
alpha_i = atan2(v_i, max(abs(u_i), V_min))
F_lat_raw_i = - C_alpha_i * alpha_i
```

再用摩擦限制饱和。但由于文件没有提供 `C_alpha_i`，简单 A320 模型中可先省略。

## 11. 摩擦圆/摩擦椭圆限制

真实轮胎的纵向刹车力和侧向力共享同一个摩擦能力，不能同时无限大。因此必须做摩擦限制。

先计算未限幅力：

```text
F_x_raw_i = F_roll_i + F_brake_i
F_y_raw_i = F_lat_raw_i
```

摩擦圆模型：

```text
F_t_max_i = mu_total * N_i
F_t_raw_i = sqrt(F_x_raw_i^2 + F_y_raw_i^2)

if F_t_raw_i > F_t_max_i:
    scale = F_t_max_i / F_t_raw_i
    F_x_i = scale * F_x_raw_i
    F_y_i = scale * F_y_raw_i
else:
    F_x_i = F_x_raw_i
    F_y_i = F_y_raw_i
```

推荐：

```text
mu_total = 0.75    // 干跑道
```

更精细一点可用摩擦椭圆：

```text
q = sqrt((F_x_raw_i / (mu_x * N_i))^2 + (F_y_raw_i / (mu_y * N_i))^2)

if q > 1:
    F_x_i = F_x_raw_i / q
    F_y_i = F_y_raw_i / q
else:
    F_x_i = F_x_raw_i
    F_y_i = F_y_raw_i
```

起飞和着陆仿真建议至少使用摩擦圆，否则重刹车时还可能产生过大的侧向纠偏力，不符合实际飞机。

## 12. 单个起落架总力

在地面系中：

```text
F_i^g = N_i * n_up^g + F_x_i * e_xw_i^g + F_y_i * e_yw_i^g
```

转到机体系：

```text
F_i^b = R_g^b * F_i^g
```

平直跑道、小姿态角近似下，可直接写：

```text
F_i^b ≈ [F_x_i, F_y_i, -N_i]^T
```

但正式六自由度仿真推荐使用完整坐标转换。

## 13. 起落架合力和力矩

每个起落架对飞机质心的力矩：

```text
M_i^b = r_i^b × F_i^b
```

全机起落架合力和力矩：

```text
F_gear^b = Σ F_i^b
M_gear^b = Σ (r_i^b × F_i^b)
```

对于你的机体系，若使用近似 `F_i^b = [F_x, F_y, -N]^T`，则：

```text
M_x = y_i * (-N_i) - z_i * F_y_i
M_y = z_i * F_x_i - x_i * (-N_i)
M_z = x_i * F_y_i - y_i * F_x_i
```

物理含义：

| 力矩 | 主要来源 |
|---|---|
| `M_x` 滚转力矩 | 左右主轮支反力不一致、侧向力作用高度 |
| `M_y` 俯仰力矩 | 前轮/主轮支反力分配、刹车力作用高度 |
| `M_z` 偏航力矩 | 左右刹车不一致、侧向力、前轮转向 |

## 14. 起飞阶段要求

起飞滑跑时应满足：

1. 所有接触力只在 `delta_i > 0` 时存在。
2. 随着升力增大，起落架法向载荷自动下降。
3. 抬轮时前起落架压缩量应先减小到 0，前轮支反力消失。
4. 离地时三个起落架 `N_i = 0`，且接地点不再穿入地面。

实现检查：

```text
if delta_nose == 0:
    N_nose = 0

if delta_nose == 0 and delta_left == 0 and delta_right == 0:
    aircraft_on_ground = false
```

起飞阶段不建议使用大刹车力；如果有 RTO 中止起飞，则主轮刹车 `B_left/B_right` 增大，并用摩擦圆限制纵向和侧向力。

## 15. 着陆阶段要求

着陆接地时应满足：

1. 主起落架通常先接地，前起落架随后接地。
2. 垂向支反力由弹簧和阻尼共同产生，阻尼项在压缩阶段吸收冲击。
3. 不能让阻尼在伸长阶段产生向下拉地面的力，因此 `N_i` 必须非负。
4. 纵向刹车力只在主起落架有法向载荷时产生。
5. 侧向力和刹车力必须共享摩擦极限。

接地冲击速度参考源文件：

| 起落架 | 冲击损坏速度阈值 |
|---|---:|
| 前起落架 | 4.064 m/s |
| 主起落架 | 6.096 m/s |

可用检查：

```text
if touchdown_vertical_speed_i > impact_threshold_i:
    hard_landing_or_damage_flag = true
```

该阈值用于损伤/告警，不建议直接作为支反力限幅。

## 16. 跑道摩擦参数建议

A32NX 文件没有提供真实轮胎摩擦系数，因此需要按跑道条件设定：

| 跑道条件 | mu_brake | mu_lat | mu_roll |
|---|---:|---:|---:|
| 干燥跑道 | 0.60 到 0.80 | 0.70 到 0.85 | 0.010 到 0.020 |
| 湿跑道 | 0.30 到 0.45 | 0.35 到 0.55 | 0.012 到 0.025 |
| 积水/污染跑道 | 0.15 到 0.30 | 0.20 到 0.35 | 0.020 到 0.040 |
| 结冰跑道 | 0.05 到 0.15 | 0.05 到 0.20 | 0.010 到 0.030 |

源文件中的：

```text
ground_high_speed_steeringwheel_static_friction_scalar = 1
ground_high_speed_otherwheel_static_friction_scalar = 1
```

可作为摩擦缩放因子使用。当前均为 1，因此不改变上述摩擦系数。

## 17. 起落架气动影响

放下起落架还会增加阻力和俯仰力矩：

```text
CD = CD_clean + gear_down * 0.0372
Cm = Cm_clean + gear_down * 0.0022
```

其中：

| 参数 | 数值 |
|---|---:|
| `drag_coef_gear` | 0.0372 |
| `pitch_moment_gear` | 0.0022 |

起飞阶段通常 `gear_down = 1`，离地后收起过程中可让 `gear_down` 从 1 线性变到 0。降落阶段放起落架后取 `gear_down = 1`。

## 18. 可省略的部分及原因

| 可省略项 | 原因 |
|---|---|
| 轮心动力学 | 文件没有直接提供轮心动力学参数；若不做轮速、防滞和轮胎转动惯量，接地点模型已经足够 |
| 轮胎 Pacejka 模型 | 文件没有给侧偏刚度、纵滑刚度、峰值滑移率等参数，强行建复杂模型不可靠 |
| 防滞刹车细节 | 简化模型用摩擦圆限制即可避免超出物理摩擦极限 |
| 液压收放机构 | 起飞/降落动力学只需要起落架放下/收起状态和接触力 |
| 擦地点 point.3 到 point.8 | 只用于翼尖、机尾、机身擦地等异常工况，正常自主起降可不建 |
| 刹车温度模型 | 不影响单次起飞/着陆动力学主响应 |

## 19. 推荐最小实现伪代码

```text
for each gear i:
    r_b = gear_contact_position_body[i]

    p_g = p_CG_g + R_b_to_g * r_b
    v_g = v_CG_g + omega_g × (R_b_to_g * r_b)

    s = n_up_g · (p_g - p_ground_g)
    delta = clamp(-s, 0, delta_max[i])
    delta_dot = - n_up_g · v_g

    if delta <= 0 and delta_dot <= 0:
        N = 0
        F_i_g = [0, 0, 0]
    else:
        N = max(0, k[i] * delta + c[i] * delta_dot)

        v_t = v_g - (v_g · n_up_g) * n_up_g

        e_xw_g = R_b_to_g * wheel_longitudinal_axis_body(i, steering_angle)
        e_xw_g = normalize(e_xw_g - (e_xw_g · n_up_g) * n_up_g)
        e_yw_g = n_up_g × e_xw_g

        u = v_t · e_xw_g
        v = v_t · e_yw_g

        F_roll = -mu_roll * N * smooth_sign(u)
        F_brake = -mu_brake * B[i] * N * smooth_sign(u)
        F_x_raw = F_roll + F_brake

        F_y_raw = -mu_lat * N * sat(v / V_lat_ref, -1, 1)

        // friction circle
        F_t_max = mu_total * N
        F_t_raw = sqrt(F_x_raw^2 + F_y_raw^2)
        if F_t_raw > F_t_max:
            scale = F_t_max / F_t_raw
            F_x = scale * F_x_raw
            F_y = scale * F_y_raw
        else:
            F_x = F_x_raw
            F_y = F_y_raw

        F_i_g = N * n_up_g + F_x * e_xw_g + F_y * e_yw_g

    F_i_b = R_g_to_b * F_i_g
    M_i_b = r_b × F_i_b

F_gear_b = sum(F_i_b)
M_gear_b = sum(M_i_b)
```

## 20. 推荐初始参数汇总

| 参数 | 推荐值 |
|---|---:|
| `mu_roll` | 0.015 |
| `mu_brake` | 0.65 |
| `mu_lat` | 0.75 |
| `mu_total` | 0.75 |
| `V_lat_ref` | 2.0 m/s |
| `eps_v` | 0.2 m/s |
| 前轮低速最大转向角 | 1.309 rad |
| 前轮高速最大转向角 | 0.105 rad |

最小可用力模型：

```text
N_i = max(0, k_i * delta_i + c_i * delta_dot_i)

F_x_i = -mu_roll * N_i * smooth_sign(u_i)
        -mu_brake * B_i * N_i * smooth_sign(u_i)

F_y_i = -mu_lat * N_i * sat(v_i / V_lat_ref, -1, 1)

[F_x_i, F_y_i] = friction_limit(F_x_i, F_y_i, mu_total * N_i)

F_i = N_i * n_up + F_x_i * e_xw + F_y_i * e_yw
M_i = r_i × F_i
```

这套模型满足正常 A320 简化起飞滑跑、抬轮离地、主轮接地、前轮压下、着陆滑跑、方向保持和刹车减速仿真的物理约束。
