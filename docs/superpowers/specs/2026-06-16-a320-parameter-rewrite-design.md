# A320 飞机参数重写设计

## 概述

将现有 6 个 MATLAB .m 文件中的小型无人机参数（~600kg）替换为 A320 级别参数（~42,500kg），来源为《基于A320气动数据的无人机自主起降巡航完整仿真方案》。

## 设计决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 气动模型形式 | 保持线性导数 + CL/Cm 插值表 | 兼顾精度和代码简洁 |
| 质量 | 空机 42,500.244 kg（§14.1） | 用户指定 |
| 推力 | 240,000 N（双发典型值） | 待后续精确数据替换 |
| 气动系数 | 直接使用文档 §6~§10 数值 | 标注 B/C 级待标定 |
| 起落架模型 | 保持线性弹簧阻尼 | 更新 A320 几何位置 |
| 转动惯量 | 文档 §14.1 数值 | 按标准机体系（X前Y右Z下）映射 |
| 着陆轨迹 | 保持几何轨迹法 | 不改控制器架构 |
| 构型增量 | 仅加襟翼 | 起飞着陆影响最大 |
| 俯仰率气动项 | 去掉 CLq、Cmq | 用户指定 |
| 控制器设计点 | 双设计点 | 巡航(234m/s) + 起飞着陆(118.8m/s) |

## 文件改动

### basedata.m（重写）

**删除的参数：**
- CLq, Cmq（俯仰率对升力/力矩影响不建模）
- CLa, Cma 单值（替换为插值表）

**新增的参数：**
- `alpha_CL_table`, `CL_table`：§7 的 9 节点 CL(α) 查表数据
- `alpha_Cm_table`, `Cm_table`：§8 的 14 节点 Cm(α) 查表数据
- `Delta_CL_flap = 1.867`：襟翼增升系数（§6）
- `Delta_CD_flap = 0.1316`：襟翼增阻系数（§6）
- `Delta_Cm_flap = -0.084`：襟翼俯仰力矩增量（§6）
- `CL_max_flap`：从全襟翼失速速度反推（§12）
- `VFE = 140.97 m/s`：放襟翼最大速度（§12）
- `thrust_tau = 1.5 s`：推力一阶时间常数（估算）
- `Ixz = 1355.818`：耦合惯量（§14.1）

**保留不变的参数：**
- 起落架刚度/阻尼：保留线性弹簧阻尼模型，刚度按质量比例缩放（k_new = k_old × 42500/600 ≈ k_old × 70.8），阻尼按 sqrt(质量比) 缩放以保持相近的阻尼比
- 风模型：保留有色噪声结构
- WOW 逻辑：保留简化的压缩量判断

**Simulink 中的 CL/Cm 计算方式：**
```matlab
CL_base = interp1(alpha_CL_table, CL_table, alpha, 'pchip');
Cm_base = interp1(alpha_Cm_table, Cm_table, alpha, 'pchip');
CL = CL_base + CLde * delta_e + Delta_CL_flap * flap_ratio;
Cm = Cm_base + Cmde * delta_e + Delta_Cm_flap * flap_ratio;
```

### control_design.m（重算）

保持原公式结构。两个设计点分别计算增益：

**设计点 1：巡航（V=234.07 m/s）**
- 由 CL_trim = mg/(qc·S) 反查配平 α
- 从 CL 表和 Cm 表提取该点的局部斜率 CLa_local、Cma_local
- 代入原公式计算 Kw_pitch, K_alpha, Kn_pitch, Kw_yaw, K_beta, Kn_yaw

**设计点 2：起飞/着陆（V=118.84 m/s）**
- 同上流程
- 输出第二套增益

输出两套增益到 `control_design.mat`，Simulink 中根据飞行阶段切换。
