# 轮式起降无人机建模与仿真

基于 MATLAB / Simulink 的固定翼无人机**轮式起降**全流程仿真平台，涵盖六自由度非线性动力学、起落架地面力学、多阶段轨迹规划与多通道飞行控制。

## 项目概览

| 属性 | 说明 |
|---|---|
| 机型 | 固定翼无人机（600 kg） |
| 起降方式 | 轮式滑跑起降（前三点式起落架） |
| 动力学 | 六自由度非线性刚体动力学 |
| 控制器 | 纵/横/航向解耦控制 + 油门/刹车控制 |
| 环境 | 有色噪声风扰动模型 |
| 工具 | MATLAB R202X + Simulink + Stateflow |

## 飞行阶段

```
起飞：  地面滑跑(1) → 抬轮(2) → 离地(3) → 初始爬升(4) → 巡航(5)

降落：  进场定高(6) → 轨迹捕获(7) → 陡下滑(8) → 圆弧拉平(9)
         → 浅下滑(10) → 指数拉飘(11) → 接地(12) → 滑跑刹车(13)
```

### 子系统说明

**6DOF-MODEL（六自由度动力学模型）**
- `AERO-PARAMETER`：气动力系数计算（升力、阻力、侧力、俯仰/滚转/偏航力矩）
- `动力学`：包含力方程、力矩方程、运动学方程、高度计算
- `起落架模型`：三轮独立弹簧-阻尼-摩擦模型，含法向支反力、纵向力（刹车/滚动阻力）、侧向力，摩擦圆限幅

**Flight Phase Manager（飞行阶段管理器）**
- Stateflow 状态机实现 14 阶段自动切换
- 数据预处理：起飞判断（到达抬轮速度、离地、安全爬升等）和着陆判断

**Guidance（制导系统）**
- 起飞制导：地面滑跑偏航指令 → 抬轮/爬升指令 → 巡航高度保持
- 降落制导：5 段着陆轨迹（定高平飞 → 陡下滑 → 圆弧拉平 → 浅下滑 → 指数拉飘）

**Controller（控制器）**
- 俯仰通道：234 段俯仰角控制 / 567 段三回路（法向过载 + 俯仰角速率 + 伪攻角）/ 8-12 段着陆俯仰控制
- 偏航通道：侧偏距 + 偏航角阻尼
- 滚转通道：滚转角指令跟踪
- 动力通道：速度 PI 控制 + 油门前馈
- 执行器限幅：升降舵 ±25°、副翼 ±25°、方向舵 ±25°、推力 0~3000 N

## 着陆轨迹设计

着陆段采用 5 段几何拼接轨迹：

```
② 陡下滑  →  ③ 圆弧拉平  →  ④ 浅下滑  →  ⑤ 指数拉飘  →  接地
   γ₁ ≈ -4°     R = 8000 m      γ₂ ≈ -1.5°     σ = 6 s
```

- **γ₁（陡下滑角）**：由怠速推力与阻力平衡推导
- **γ₂（浅下滑角）**：由接地下沉率约束推导
- **圆弧段**：以固定半径 R 平滑过渡两段下滑线
- **拉飘段**：指数衰减高度曲线，确保柔和接地

## 使用方法

### 环境要求
- MATLAB R2020b 或更高版本
- Simulink
- Stateflow

### 运行步骤

1. **加载参数**

   在 MATLAB 命令窗口运行：
   ```matlab
   plane_int
   ```
   该脚本依次加载飞机参数、控制器设计、着陆轨迹和起飞参数。

2. **打开模型**

   ```matlab
   open_system('takeoffANDlandingV3')
   ```

3. **运行仿真**

   在 Simulink 中点击 **Run**，或执行：
   ```matlab
   sim('takeoffANDlandingV3')
   ```

4. **查看结果**

   仿真结束后，双击模型中的 `plot` 子系统查看各状态曲线，或在工作区中自定义绘图。

### 着陆轨迹预览

单独运行 `landing_trajectory.m` 可生成着陆轨迹几何设计图，包括轨迹剖面图和圆弧放大图。

## 坐标系说明

采用**前-右-下**（Forward-Right-Down）机体/地面坐标系：

| 轴 | 方向 |
|---|---|
| X | 向前（机头方向） |
| Y | 向右 |
| Z | 向下 |

状态量：`x = [Xg; Yg; Zg; u; v; w; φ; θ; ψ; p; q; r]`

- `Xg, Yg, Zg`：地面系位置
- `u, v, w`：机体系速度分量
- `φ, θ, ψ`：欧拉角（滚转、俯仰、偏航）
- `p, q, r`：机体系角速率

## 风扰动模型

采用有色噪声（Ornstein-Uhlenbeck 过程）模拟真实风扰动：

| 相关时间常数 (s) | 10 | 10 | 10 |
| 基准风速 (m/s) | 0 | 5 | 0 |
